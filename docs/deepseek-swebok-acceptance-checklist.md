# DeepSeek 与 SWEBOK 链路人工验收清单

本文用于人工验收真实 DeepSeek、SWEBOK PDF、Chroma 索引和 AI 岗位问卷链路。Agent 不执行真实 key 验收、不下载模型、不判断最终产品效果。

## 验收前置条件

| 检查项 | 状态 | 备注 |
| --- | --- | --- |
| `DEEPSEEK_API_KEY` 已在服务器环境变量中配置 | 未验收 / 通过 / 失败 | 不写入前端 `.env`，不提交 git。 |
| `rag-spike/private-data/swebok-v4.pdf` 存在 | 未验收 / 通过 / 失败 | 私有 PDF 不提交 git。 |
| Chroma 索引已构建 | 未验收 / 通过 / 失败 | 检查 `rag-spike/outputs/index-build-report.json`。 |
| `document_count > 0` | 未验收 / 通过 / 失败 |  |
| `pdf_extractable_pages > 0` | 未验收 / 通过 / 失败 |  |

## AI 岗位问卷检查项

- [ ] 返回 15 道题。
- [ ] 每题包含合法 `capability_key`。
- [ ] 每题包含 `role_dimension_id`。
- [ ] 15 题覆盖全部 6 个岗位维度。
- [ ] 题目能提交到同一个 `assessment_id`。
- [ ] 提交后进入个人界面，显示个人能力雷达和能力明细。

## 失败路径检查项

| 场景 | 期望结果 | 状态 |
| --- | --- | --- |
| 缺少 `DEEPSEEK_API_KEY` | 后端返回清晰错误，前端提示配置问题，不生成 mock 报告 | 未验收 / 通过 / 失败 |
| RAG / DeepSeek 失败 | 前端显示无法生成岗位能力模型或 AI 问卷，不 mock | 未验收 / 通过 / 失败 |
| SWEBOK PDF 缺失或索引为空 | AI 岗位问卷失败并提示知识库或索引问题 | 未验收 / 通过 / 失败 |
| Ability scoring 失败 | 不生成 mock 个人雷达，前端显示错误 | 未验收 / 通过 / 失败 |
| `assessment_id` 不属于当前学生 | 返回 404 或明确错误 | 未验收 / 通过 / 失败 |
