# 真实样例主流程记录模板

本文用于记录人工使用真实或高可信脱敏样例重跑能力评估主流程的结果。Agent 不负责收集真实样例、不执行真实 DeepSeek 链路验收、不判断产品效果是否通过；Agent 只维护模板、字段和问题回填格式。

## 使用边界

- 不提交真实手机号、邮箱、证件号、住址、学号、精确生日。
- 不使用“姓名 XX / 岗位 XX / 公司 XX”作为主演示数据。
- 真实简历、真实 JD、真实公司信息必须经过脱敏或取得测试授权。
- 真实效果判断由人工完成，包括岗位维度是否合理、报告建议是否可执行、输出是否可用于演示。
