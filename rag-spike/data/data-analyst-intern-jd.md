# 数据分析实习生 JD

## 岗位名称

数据分析实习生

## 岗位背景

团队希望通过数据分析理解学生求职行为、岗位浏览行为、简历准备进度、活动参与情况和智能体使用效果，为产品迭代和运营决策提供依据。

## 工作职责

1. 协助整理业务数据，包括用户行为数据、问卷数据、活动报名数据和岗位浏览数据。
2. 使用 Excel、SQL 或 Python 完成基础数据清洗、统计和指标计算。
3. 根据业务问题设计分析思路，例如学生使用路径、功能转化、活跃度和留存情况。
4. 输出分析报告或可视化图表，帮助产品、运营和就业服务团队理解数据结论。
5. 与业务同学沟通分析口径，确认指标定义、样本范围和数据限制。
6. 基于数据结果提出产品或运营优化建议。

## 任职要求

1. 对数据分析、互联网产品、教育科技或 AI 应用方向有兴趣。
2. 具备较好的逻辑分析能力，能够围绕业务问题拆解指标。
3. 熟悉 Excel，了解 SQL 或 Python 者优先。
4. 具备基础统计思维，能理解均值、比例、趋势、分组对比和异常值。
5. 具备清晰表达能力，能用业务同学能听懂的方式解释数据结论。
6. 做事认真细致，有较强执行力和数据质量意识。

## 能力观察重点

- 数据与数字化思维：能否使用工具和数据指标完成分析。
- 逻辑分析能力：能否把业务问题拆成指标和分析路径。
- 沟通表达能力：能否解释分析结论和数据限制。
- 执行推进能力：能否稳定完成清洗、统计和报告输出。
- 学习适应能力：能否快速学习新的数据口径、工具和业务背景。
- 商业与行业理解：能否理解数据结论对产品或运营决策的意义。
